# 安卓视频播放器   

## 功能  
 
1. 实现最简单http服务器->手机扫码发送视频链接->服务端复制视频链接，并尝试调起系统播放器；  

## 使用方法   
1.  [从releases下载最新版本](https://github.com/qidizi/android-tv-player/releases);    
1.  安装至安卓电视;   
1.  运行本应用;  
1.  微信扫码后使用;  

## 正常使用的环境   
1.  avd android api 29  
1.  坚果投影仪m6
