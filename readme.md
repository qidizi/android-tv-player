# 安卓视频播放器   

## 功能  
 
1. 最简单http服务器->手机扫码发送视频页面的链接->tv端指定以firefox打开链接->firefox以插件方式实现操作视频自动播放    

## 使用方法   
1. [从releases下载最新版本](https://github.com/qidizi/android-tv-player/releases);    
2. 安装至安卓电视;   
3. 运行本应用;  
4. 微信扫码后使用;  

## 正常使用的环境   
1. avd android api 29  
2. 坚果投影仪m6 
3. 米9 miui 12
